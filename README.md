1. create-react-app을 사용해 프로젝트 생성
> npx create-react-app mini-blog

2. 프로젝트 디렉터리에 들어가서 앱 실행하기
> cd mini-blog
> npm start

3. 패키지 설치하기
> npm install --save react-router-dom styled-components

4. Production 빌드하기
[코드와 애플리케이션이 사용하는 이미지,CSS 파일 등의 파일을 모두 모아서 패키징 하는 과정]
> npm run build

5. serve 설치하기
> npm install -g serve

6. serve 빌드하기
> serve -s build
